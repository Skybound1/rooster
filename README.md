# ![Logo](https://github.com/thomasleese/rooster/raw/master/rooster/static/images/logo.png)

An event scheduler.

## Setup 

    $ pyvenv venv
    $ source venv/bin/activate
    $ pip install -r requirements.txt
    $ python manage.py migrate

## Running

    $ source venv/bin/activate
    $ python manage.py runserver

## When done

    $ deactivate
